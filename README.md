# 🛡️ Oura VPN — High-Speed VLESS Client for Windows

**Oura VPN** — это современный и быстрый VPN-клиент для Windows, построенный на базе Flutter и ядра Xray. Приложение создано для тех, кто ищет максимальную скорость, обход блокировок и стильный интерфейс в одном флаконе.

## ✨ Основные фишки
*   **Протоколы нового поколения:** Использование VLESS/Xray обеспечивает стабильное соединение и высокую скорость, достаточную для 4K-стриминга и онлайн-игр.
*   **Flutter UI:** Плавные анимации, современный «темный» дизайн и интуитивно понятный пользовательский опыт.
*   **Умная проверка прав:** Приложение автоматически проверяет наличие прав администратора, необходимых для корректной настройки сетевых маршрутов и работы драйверов.
*   **Полная приватность:** Мы не ведем логи активности. Ваша история просмотров остается только вашей.

## 💳 Инновационная система оплаты
Oura VPN поддерживает самые современные способы подписки, интегрированные напрямую в экосистеме мессенджеров и блокчейна:
1.  🌟 **Telegram Stars** — быстрая оплата внутри Telegram.
2.  💎 **TON (Crypto)** — анонимные платежи через блокчейн.
3.  💳 **YooMoney** — стандартная оплата банковскими картами.

## 🚀 Как запустить (для пользователей)
1. Скачайте последнюю версию из раздела [Releases](https://github.com/ваш-репозиторий/releases).
2. Распакуйте архив и запустите `oura_vpn.exe`.
3. **Важно:** Для работы VPN-протоколов приложение необходимо запускать **от имени администратора**.

## 🛠 Технологии
*   **Frontend:** Flutter
*   **Backend logic:** Dart, Xray Core
*   **Database:** Firebase (Firestore / Auth)
*   **Payment Integration:** Telegram Bot API, TON API
